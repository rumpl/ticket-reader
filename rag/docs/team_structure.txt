# Acme Corp — Team Structure & Organization

## Engineering Organization
**VP Engineering:** Pat Kumar | **CTO:** Raj Patel | **Total Engineering:** 142

### Platform Team (18 engineers)
**Lead:** Alex Rivera (SRE Lead)
- **SRE Squad** (6): Infrastructure reliability, on-call, incident response
- **Platform Squad** (7): Kubernetes, CI/CD, developer experience, internal tooling
- **Security Squad** (5): AppSec, compliance automation, vulnerability management

**Responsibilities:**
- Maintain shared Go libraries and service templates
- Own CI/CD pipeline and deployment infrastructure
- Manage Kubernetes clusters, Kafka, databases
- Run chaos engineering and DR exercises
- Provide golden path for new service creation (<30 min to first deploy)

### Product Engineering (96 engineers, 8 squads)

#### Supply Chain Domain (4 squads)
- **Inventory Squad** (12): Real-time inventory tracking, warehouse integrations
- **Procurement Squad** (12): Purchase orders, supplier management, approval workflows
- **Logistics Squad** (12): Shipping, tracking, carrier integrations (FedEx, UPS, DHL)
- **Forecasting Squad** (12): Demand prediction ML models, planning tools

#### Analytics Domain (2 squads)
- **Insights Squad** (12): Dashboards, custom reports, data export
- **Data Pipeline Squad** (12): ETL, ClickHouse, data warehouse, batch processing

#### Collaboration Domain (2 squads)
- **Connect Squad** (12): SupplyView Connect (supplier portal), messaging
- **Integration Squad** (12): Webhooks, API platform, EDI, third-party connectors

### Architecture & Staff Engineers (8)
- **Chief Architect**: Maria Santos
- Review all ADRs and cross-team technical decisions
- Maintain system design documentation
- Lead quarterly architecture reviews
- Mentor senior engineers toward staff/principal track

### Engineering Managers (20)
- Each squad has a dedicated EM
- EM:engineer ratio target is 1:6
- EMs do not write production code (focus on people and process)
- Weekly EM sync led by VP Engineering
