## Procurement Process (PROC-POL-002)

### Spending Thresholds
- **< $5,000/year**: Manager approval, no procurement involvement
- **$5,000 – $25,000/year**: Manager + Director approval, procurement review
- **$25,000 – $100,000/year**: VP approval, procurement-led negotiation, legal review
- **> $100,000/year**: C-level approval, board notification, competitive RFP required

### Vendor Evaluation Criteria
All new vendors scoring above $25,000/year must be evaluated on:
1. **Security posture** — SOC 2 Type II required, ISO 27001 preferred
2. **Data residency** — Must support EU data residency for GDPR compliance
3. **SLA** — Minimum 99.9% uptime guarantee
4. **Exit strategy** — Data export capability and contractual termination terms
5. **Integration** — API availability, SSO/SAML support

### Security Review
Vendors handling Confidential or Restricted data (see POL-SEC-002) must complete:
- Security questionnaire (SIG Lite or equivalent)
- Penetration test results (within last 12 months)
- DPA (Data Processing Agreement) signed
- Annual review by Security team
